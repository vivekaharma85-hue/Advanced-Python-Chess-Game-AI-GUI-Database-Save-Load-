# ♟️ Full Chess Game with AI (Python + Tkinter)

A feature-rich desktop Chess application built with **Python** and
**Tkinter**. It includes a responsive UI, AI opponent, save/load
support, SQLite database integration, customizable piece styles, and
complete chess rules.

## Features

-   ✅ Complete Chess Rules
-   🤖 AI Opponent (Minimax + Alpha-Beta Pruning)
-   🎨 Responsive Tkinter GUI
-   💾 Save & Load Games
-   🗄️ SQLite Database for Game History
-   ↩️ Undo Moves
-   👑 Pawn Promotion
-   🏰 Castling
-   🎯 En Passant
-   🔁 Threefold Repetition Detection
-   ⏱️ 50-Move Rule
-   🎭 Multiple Piece Styles
-   🎨 Custom Piece Colors
-   📜 Game History Viewer

## Technologies Used

-   Python 3
-   Tkinter
-   SQLite3
-   Pickle
